Getting Started
===============

To understand how to use Scanner, check out the :ref:`quickstart`, which
talks through the components of the system with a simple example.

Once you have a handle on the basics, you can checkout :ref:`walkthrough`,
which provides a more full fledged walkthrough of using Scanner in a real
application, or :ref:`tutorial`, which provides examples of many of the
features of the API.
